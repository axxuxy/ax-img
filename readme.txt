由于Y站国内无法正确解析到ip地址，要访问的话需要设置hosts文件。

会修改hosts文件的话直接添加 9-10 行的hosts记录 不会的话继续往下读。

打开  C:\Windows\System32\drivers\etc  目录  用记事本或其他文本编辑工具  打开目录下的hosts文件。

添加以下8-10行 记录 至末尾：

198.98.54.92 yande.re
198.98.54.92 assets.yande.re
198.98.54.92 files.yande.re

保存对hosts文件的修改 ( 按Ctrl+s 或者 点击编辑工具 文件 -> 保存 选项 )

这可能会遇到无法保存需要管理员权限无法保存的情况。这时你可以保存到其他目录再复制过来替换该文件。这时可能要你提供管理员权限。确认就好。
